# Online Bidding Management System (DBMS Project)

## 📌 Project Description

This project is an Oracle SQL-based Online Bidding Management System developed as part of the Database Management Systems course.

It allows users to list products, participate in auctions, place bids, and automatically determines winners based on the highest bid.

---

## 🎯 Features

* User management (buyer/seller roles)
* Product listing and categorization
* Auction creation with time constraints
* Bid validation using triggers
* Automatic winner selection
* Reporting using SQL views

---

## 🗂️ Database Schema

The system consists of the following tables:

* Users
* Category
* Product
* Auction
* Bid
* Auction_Winner
